# yadm Bootstrap Setup

## Prerequisites

- Git

## Usage

1. `brew install yadm` (macOS) or equivalent
1. `yadm clone https://github.com/tifandotme/dotfiles.git` (don't bootstrap yet)
1. `yadm decrypt` (see Bitwarden notes for password)
1. `yadm bootstrap`

## One-liner

```bash
brew install yadm && yadm clone https://github.com/tifandotme/dotfiles.git && yadm decrypt && yadm bootstrap
```
